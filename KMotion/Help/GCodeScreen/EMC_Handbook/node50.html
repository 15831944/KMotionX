10.5 Axis Words

We have already seen examples of axis words. An X word would be X10.001, which by itself indicates the X axis should move to a position of 10.001 user units, which would normally be inches or mm. Table one lists the common names for axis. Not all of these names can be used by the EMC interpreter. X, Y, and Z words will be accepted, as will A, B, and C.

X Primary Linear Axis
Y Primary Linear Axis
Z Primary Linear Axis
U Secondary axis parallel to X
V Secondary axis parallel to Y
W Secondary axis parallel to Z
A Angular axis around X axis
B Angular axis around Y axis
C Angular axis around Z axis
